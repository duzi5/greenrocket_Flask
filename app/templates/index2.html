Um pequeno glossário

O que é Machine Learning O que é Data Science

É uma aplicação da inteligência artificial, consistindo em algoritmos que produzem conhecimento para uma máquina adequar suas tarefas. Um algoritmo de ML coleta um conjunto de dados de entrada, identifica padrões e gera saídas. As entradas de dados possuem seus próprios recursos e a abundância desses recursos é fundamental para qualquer algoritmo de ML. Nossos sistemas são arquitetados para potencializar a obtenção das features das entradas de dados, ao ponto de se aproveitar da inteligência criada para gerenciar as saídas de dados que estão por todo o sistema.

Data Science, ou em português, ciência de dados é uma ferramenta que se baseia no estudo disciplinado dos dados e informações a partir da análise de dados. Para isso, essa ciência foca na captura de informações, na transformação desses dados, e até mesmo, em sua geração. Data Science se apropria das ferramentas computacionais, da estatística, lógica matemática e do conhecimento do Negócio para produzir conhecimento, o qual pode ser utilizado nas mais diversas formas, sendo uma forte aliada para a condução de qualquer tipo de negócio.
